# GALOHOo | AI-first game

**GALOHOo** is a browser-based 2D pixel art game with a top-down view, built as an AI-first game development experiment.

The core idea of the game is to turn real-life productivity into in-game progress. The player creates tasks in a ToDo list, sets their importance, completes them in real life, and receives in-game gold and resources as a reward. These rewards are then used to develop a small island settlement.

## Concept

GALOHOo combines a productivity tool with a cozy colony-simulation game.

Instead of completing abstract quests, the player progresses by completing their own real tasks. The ToDo list becomes the main source of income, materials, and long-term development inside the game world.

The game is set on a randomly generated island. Small AI-controlled villagers live there, move around, collect resources, rest, build, and interact with the world. The player's goal is to grow the settlement through real-life productivity and in-game management.

## Features

- Browser-based 2D pixel game
- Top-down island view
- Random island generation on first launch
- ToDo list as the main gameplay mechanic
- Task importance system
- Rewards for completed real-life tasks
- Gold and resource economy
- Shared settlement storage with resources like wood and stone
- AI-controlled villagers with individual behavior
- Villagers collect resources and contribute to the settlement
- Randomly generated villager profiles
- Villager professions based on their actual activities
- Energy system: villagers work, get tired, and go to sleep
- Day/night cycle
- Weather system, including rain
- Settlement development through ToDo-based quests
- Era progression system that changes buildings and villager visuals
- Villagers can suggest buildings and ask the player for permission before construction
- Traveling merchant who occasionally arrives by sea
- Gacha system for spending extra gold
- Unlockable custom cursors from cases
- Interaction details such as grabbing and dragging villagers
- Special animations for dragging villagers and when villagers fall into water

## AI-first development

This project was created using an AI-first development workflow.

The codebase was mainly generated with Google Gemini. My role in the project was focused on:

- Game concept and feature design
- System logic and gameplay direction
- Prompt engineering
- Iterative feature planning
- UI/UX decisions
- Testing and bug reporting
- Small manual HTML/CSS adjustments
- Polishing the overall experience

The goal of this project was not only to create a playable browser game, but also to explore how far an AI-assisted workflow can go in building an experimental game prototype from idea to implementation.

## Project goal

GALOHOo is an experimental productivity game that turns everyday tasks into a game economy.

The project explores a simple question:

**Can a ToDo list feel like a living world instead of a boring checklist?**

By connecting real-life task completion with settlement progression, GALOHOo tries to make productivity more visual, rewarding, and playful.
